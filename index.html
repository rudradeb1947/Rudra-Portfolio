About Me

Hello! I'm Rudradeb Nandi, a passionate Software and Machine Learning Engineer with a drive for building intelligent, data-driven, and user-centric applications. My journey in tech began with a fascination for how algorithms work, evolving into a mission to craft impactful solutions through code and data.

I specialize in Python development, machine learning and full-stack web applications using tools like Scikit-Learn, SQL, Django, Flask, and PostgreSQL to build smart and scalable systems.
